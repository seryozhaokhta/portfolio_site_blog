Navigate Antarctica with an Interactive Coordinate Tracker

This code creates an interactive map of Antarctica that can be used to determine the geographic coordinates of any point on the map. As the user moves their cursor over the map, the program calculates the latitude and longitude of that point and displays them on the screen. There is an animated circle around the cursor that changes its size when the user clicks the mouse, indicating the selected point on the map. The code also allows users to toggle the display mode of the coordinate text to make it more legible on different backgrounds.
